# ⚙️ Asset Management System

🚧 Work in progress

A system for managing company IT assets such as laptops, monitors, and other hardware.

---

## ⚙️ Tech Stack
🟢 Express.js · 🔷 TypeScript · 🗄️MariaDB · 🔑 JWT · 🐳 Docker · 🟩 Vue.js 🎨 · Tailwind CSS (DaisyUI)
