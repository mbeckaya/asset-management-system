# ⚙️ Asset Management System

🚧 Work in progress

A system for managing company IT assets such as laptops, monitors, and other hardware.

---

## ⚙️ Tech Stack
🟢 Express.js · 🔷 TypeScript · 🗄️MariaDB · 🔑 JWT · 🐳 Docker · 🟩 Vue.js 🎨 · Tailwind CSS (DaisyUI)

---

## 🧩 Planned Features
- Authentication (JWT)
- CRUD for devices
- Assignment of assets to employees
- Warranty tracking
- Email notifications
- Search & filtering
- Dashboard with KPIs
